# Result Certification Demo

Runs a mocked V3 discovery workflow and highlights the generated
`v3_result_certification.json` and `v3_result_certification.md` artifacts.

## Run

```bash
examples/v3_0/result_certification_demo/run.sh
```

## What The Output Means

The output demonstrates platform/workflow certification for a V3 result bundle.

## What The Output Does Not Prove

Certification is not clinical validation and does not prove binding, activity,
safety, efficacy, or biomedical outcomes. It provides no protocol, synthesis
instruction, or dosing guidance.
