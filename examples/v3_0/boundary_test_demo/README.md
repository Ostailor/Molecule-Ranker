# Boundary Test Demo

Runs the V3 autonomy boundary-test validation command.

## Run

```bash
examples/v3_0/boundary_test_demo/run.sh
```

## What The Output Means

The output shows whether configured autonomy boundary checks pass for the
current software build.

## What The Output Does Not Prove

It does not prove clinical validity, biomedical outcomes, assay results, or
real-world safety. It provides no protocol, synthesis instruction, or dosing
guidance.
