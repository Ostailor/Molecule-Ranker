# Integration Dry-Run Loop

Runs V3 discovery in dry-run mode with integration visibility enabled. External
writes remain disabled.

## Run

```bash
examples/v3_0/integration_dry_run_loop/run.sh
```

## What The Output Means

The output shows integration-aware planning can be exercised without writing to
external systems.

## What The Output Does Not Prove

It does not prove external-system synchronization, record correctness, clinical
validity, or biomedical outcomes. It provides no protocol, synthesis
instruction, or dosing guidance.
