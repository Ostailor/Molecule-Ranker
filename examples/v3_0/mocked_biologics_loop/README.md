# Mocked Biologics Loop

Runs the V3 discovery command in mocked mode with the governed biologics track
enabled. Antibody generation remains disabled unless explicitly enabled by a
separate governed command.

## Run

```bash
examples/v3_0/mocked_biologics_loop/run.sh
```

## What The Output Means

The output shows biologics planning artifacts can be included in a V3 result
bundle with review gates.

## What The Output Does Not Prove

It does not prove antibody binding, safety, efficacy, manufacturability,
therapeutic value, or clinical validity. It provides no expression,
purification, immunization, wet-lab protocol, synthesis instruction, or dosing
guidance.
