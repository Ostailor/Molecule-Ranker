# molecule-ranker V3.0 Demo Workflows

These demos exercise V3.0 workflow, governance, result-bundle, certification,
and boundary-test surfaces. They are software/autonomy demos for internal
research planning only.

All checked-in examples use synthetic or mocked inputs. They do not include
real assay results, real citations, wet-lab protocols, synthesis instructions,
dosing guidance, or clinical claims.

## Demos

- `mocked_full_discovery_loop`
- `dry_run_full_discovery_loop`
- `read_only_live_small_molecule_loop`
- `mocked_biologics_loop`
- `campaign_copilot_event_loop`
- `integration_dry_run_loop`
- `result_certification_demo`
- `boundary_test_demo`

Each demo contains a README, a `run.sh` command script, and an
`expected_output_manifest.json` describing what the output means and what it
does not prove.
