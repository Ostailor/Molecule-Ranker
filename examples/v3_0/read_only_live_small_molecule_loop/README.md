# Read-Only Live Small-Molecule Loop

Runs the V3 discovery command in `read_only_live` mode for a synthetic
small-molecule planning scenario. The demo is read-only and does not perform
external writes.

## Run

```bash
examples/v3_0/read_only_live_small_molecule_loop/run.sh
```

## What The Output Means

The output demonstrates the live-mode boundary for read-only planning surfaces.

## What The Output Does Not Prove

It does not prove real-world evidence quality, binding, activity, safety,
efficacy, or clinical validity. It provides no protocol, synthesis instruction,
or dosing guidance.
