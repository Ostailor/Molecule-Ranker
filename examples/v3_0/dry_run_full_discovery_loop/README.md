# Dry-Run Full Discovery Loop

Runs the V3 `full_discovery_loop` in dry-run mode with synthetic planning input.

## Run

```bash
examples/v3_0/dry_run_full_discovery_loop/run.sh
```

## What The Output Means

The output shows the workflow can prepare planning artifacts while keeping
external writes disabled.

## What The Output Does Not Prove

It does not prove any biomedical fact, clinical validity, binding, activity,
safety, or efficacy. It provides no protocol, synthesis instruction, or dosing
guidance.
