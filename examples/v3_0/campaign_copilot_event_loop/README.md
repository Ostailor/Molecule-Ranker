# Campaign Co-Pilot Event Loop

Runs a synthetic campaign co-pilot validation scenario. The co-pilot may draft
planning recommendations, but campaign activation and stage gates remain
human-owned.

## Run

```bash
examples/v3_0/campaign_copilot_event_loop/run.sh
```

## What The Output Means

The output shows a synthetic co-pilot event can be routed through governed
autonomy validation.

## What The Output Does Not Prove

It does not approve a campaign, prove clinical validity, prove biomedical
outcomes, or provide a protocol, synthesis instruction, or dosing guidance.
