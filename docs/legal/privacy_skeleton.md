# Privacy Skeleton

This is a productization skeleton, not a final privacy policy.

## Data Collected

The pilot app may collect account details, organization details, project metadata,
run inputs, usage records, notes, favorites, result bundle artifacts, feedback,
and operational logs.

## Sensitive Data Boundaries

Users should not submit patient data. Molecule Ranker is for research use only and is not medical advice, not clinical decision support, and not a regulated medical product.

Secrets, credentials, API keys, access tokens, and private integration settings
must never be exposed in user-facing pages, exports, or API responses.

## Processing Purpose

Data is processed to provide project workspaces, discovery runs, ranked
candidates, evidence and provenance review, generated computational hypotheses,
exports, usage limits, support, and security operations.

## Review and Compliance

Evidence and provenance should be independently reviewed. Users are responsible
for complying with laws, institutional policies, and lab safety requirements.
